# Currency Converter

A modern, fast, and user-friendly currency converter built with **Next.js**, **Tailwind CSS**, and **ExchangeRate-API**. This application features a custom-built searchable dropdown, keyboard navigation, and real-time validation.

## Features

- **Real-time Conversion**: Fetching the latest exchange rates via a secure API.
- **Searchable Selects**: Custom dropdown components that allow users to search through 150+ currencies easily.
- **Keyboard Navigation**: 
  - Use **Arrow Up/Down** to navigate the currency list.
  - Press **Enter** to select a currency or format the amount.
- **Smart Validation**: Instant error messaging for invalid inputs (letters or negative numbers).
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop screens using Tailwind CSS.
- **Smart UX**: Includes "Click-outside" logic to close menus and automatic scrolling for active list items.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **State Management**: React Hooks (`useState`, `useEffect`, `useRef`)
- **API**: ExchangeRate-API

## Getting Started

### Prerequisites

- Node.js 18.x or later
- NPM or Bun
